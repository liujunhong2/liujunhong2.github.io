常用指令

建站前

在gitbase中cd到博客文件夹

下载hexo框架

初始化博客项目（若已经在下一级文件夹中可不加folder）

进入folder

安装依赖项

下载部署器

之后可随时部署到Github上

建站后

new 新建一篇文章

-p 自定义文章路径

-r 如果存在则替换

-s 文章别名

clean 清除缓存文件 (db.json) 和已生成的静态文件 (public)

generate 生成静态文件

publish 发布草稿

server 启动服务器

-p 重设端口

-s 只使用静态文件

-l 启用日志

deploy 部署网站

render 渲染文件
-o output destination

migrate 从其他博客系统迁移内容

list 列出所有路由

version 显示版本信息

config 列出网站的配置（_config.yml）。 如果指定了 key，则只展示配置中对应 key 的值；如果同时指定了 key 和 value，则将配置中对应的 key 的值修改为 value。
